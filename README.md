# MetaCrypt

Compile-time encryption and decryption

## Usage
Call
```
./build.sh
```
You should get the result as an error message, like this:
```
main.cpp:6:76: error: implicit instantiation of undefined template
      'The_message_is<S<'H', 'E', 'J'> >'
```
